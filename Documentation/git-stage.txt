git-stage(1)
============

NAME
----
git-stage - Add file contents to the staging area

SYNOPSIS
--------
[verse]
'git stage' [options] [--] [<pathspec>...]

DESCRIPTION
-----------

The staging area is a location where changes are stored in preparation for a commit.

This is a synonym for linkgit:git-add[1].

--------
linkgit:git-add[1]
