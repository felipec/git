git-unstage(1)
==============

NAME
----
git-unstage - Remove changes from the staging area

SYNOPSIS
--------
[verse]
'git unstage' [options] [--] [<pathspec>...]

DESCRIPTION
-----------

Removes changes from the staging area. This is the same as `git reset`.
